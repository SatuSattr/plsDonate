# Customization & Design

Make **plsDonate-Express** feel like a native part of your server by customizing its messages, sounds, and visual templates.

## 1. Localization (`lang/en-US.yml`)

Every message shown to players can be edited. We support **MiniMessage** (Adventure) for modern formatting.

### Placeholders
You can use various placeholders like `{PLAYER}`, `{AMOUNT_FORMATTED}`, and `{METHOD_COLORED}` in your messages.

**Example: Customizing the broadcast**
```yaml
donation-notification:
  - ""
  - "  <#FFD700>★ <yellow>{PLAYER} <white>just supported the server with <green>Rp{AMOUNT_FORMATTED}!"
  - "  <gray>Message: <italic>\"{MESSAGE}\""
  - ""
```

## 2. Global Number Formatting
Set your currency format in `config.yml` to match your region.

```yaml
format:
  # id-ID: 1.000.000 (Indonesian)
  # en-US: 1,000,000 (US English)
  locale: "id-ID"
```

## 3. Audio Feedback (`sound-effects`)
The plugin plays sounds during different stages of the donation.

Format: `"sound_name,pitch,volume"`
- `donation-confirmation`: Played when the `/donate` prompt appears.
- `donation-received`: Played to **all** players when a donation is successful.

```yaml
sound-effects:
  donation-received:
    - "minecraft:entity.player.levelup,1.2,1.0"
    - "minecraft:block.note_block.chime,1.0,1.0"
```

## 4. Email Template (`templates/payment.html`)
The email sent to players is a standard HTML file found in `plugins/plsDonate-Express/templates/`.

- You can use standard HTML and CSS to style your email.
- **Placeholders:** `{PLAYER}`, `{AMOUNT_FORMATTED}`, `{LINK}`, `{METHOD}`, `{MESSAGE}`.
