# Getting Started

Installing **plsDonate-Express** is straightforward. Follow this guide to ensure your environment is ready for secure donation processing.

## Prerequisites

Before installing the plugin, ensure your server meets the following requirements:

- **Minecraft Version:** Paper, Spigot, or Purpur 1.21 or newer.
- **Java Version:** Java 21 is required for high-performance networking features.
- **Public IP/Domain:** Your server must have a public IP or domain and the ability to open a port (default: `21172`) for the webhook listener.

## Required Dependencies (Optional but Recommended)

While the core plugin works standalone, these dependencies unlock its full potential:

1.  **Floodgate & Geyser:** Required for **Bedrock Edition** native UI support.
2.  **PlaceholderAPI:** Required to display donation statistics and goals in chat or on holograms.
3.  **An Economy Plugin:** (e.g., EssentialsX, Vault) if you plan to give money as a reward.

## Installation Steps

1.  **Download the Plugin:** Place the `plsDonate-Express-1.0.jar` into your server's `plugins/` folder.
2.  **Restart your Server:** Restart (or start) the server to generate the default configuration files.
3.  **Check the Console:** On the first run, you will see a message indicating the plugin has loaded. You will also see warnings if your API keys are not yet configured.

[Screenshot: Server console showing the successful startup message for plsDonate-Express]

4.  **Initial Configuration:** Navigate to `plugins/plsDonate-Express/config.yml` to begin setting up your [Tako.id integration](./configuration/tako-setup).

---

### Need help?
If you encounter any issues during installation, check your `latest.log` for error messages. Common issues are usually related to Java versions (ensure you are on 21) or port conflicts with the webhook listener.
